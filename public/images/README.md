# Slider Images

Эта папка содержит PNG иконки для слайдера на главной странице.

## Файлы:

- `slide1.png` - Оранжевый слайд "Telegram Mini App за 14 дней"
- `slide2.png` - Фиолетовый слайд "Скидка 15% на разработку бота"
- `slide3.png` - Синий слайд "Автоматизация бизнес-процессов"
- `slide4.png` - Зелёный слайд "Бесплатная консультация"

## Требования к изображениям:

- **Формат**: PNG с прозрачным фоном
- **Рекомендуемый размер**: 120x120px или больше
- **Стиль**: Светлые иконки с эффектами свечения, бликов, звездочек
- **Цвета**: Белый как основа + цветные акценты
